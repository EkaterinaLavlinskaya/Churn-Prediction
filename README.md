# Прогнозирование оттока клиентов (Churn Prediction)

## 📌 Описание проекта
Этот проект посвящен построению модели машинного обучения для прогнозирования оттока клиентов (Churn) на основе демографических данных и данных об использовании услуг. Задача — бинарная классификация: предсказать, уйдет клиент (1) или останется (0).

Проект демонстрирует полный цикл работы с данными: от загрузки и разведочного анализа (EDA) до обучения модели и оценки ее качества.

## 📊 Используемые данные
- Датасет содержит информацию о клиентах: период обслуживания, ежемесячные траты, общая сумма трат, демографические данные (пол, возрастная группа, семейное положение), а также информация о подключенных услугах.
- Данные были загружены с Google Drive в формате CSV.

## 🛠️ Стек технологий
- **Python** (основной язык)
- **Pandas, NumPy** (обработка данных)
- **Matplotlib** (визуализация)
- **Scikit-learn** (модели и предобработка)
  - `StandardScaler` (нормализация числовых признаков)
  - `OneHotEncoder` (кодирование категориальных признаков)
  - `LogisticRegression` (модель классификации)
  - `train_test_split`, `classification_report`, `confusion_matrix`, `roc_auc_score`
