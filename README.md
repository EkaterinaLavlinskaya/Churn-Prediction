# Прогнозирование оттока клиентов (Churn Prediction)

## 📌 Бизнес-задача
Клиенты телеком-компании уходят к конкурентам, что приводит к прямой потере выручки. Цель проекта — разработать модель машинного обучения, которая по историческим данным (демография, подключённые услуги, длительность обслуживания) предскажет вероятность оттока для каждого клиента. Это позволит бизнесу своевременно применять удерживающие мероприятия (персональные скидки, специальные предложения) и сократить потери.

## 📊 Данные
*   **Источник:** Kaggle (датасет по оттоку клиентов)
*   **Размер:** 5282 строки, 20 признаков (числовые и категориальные).
*   **Целевая переменная:** `Churn` (1 — клиент ушёл, 0 — остался).

## 🔍 Что сделано
*   **Разведочный анализ (EDA):** Визуализация распределений признаков, анализ взаимосвязей с целевой переменной, выявление ключевых факторов, влияющих на отток (например, тип контракта, способ оплаты, длительность обслуживания).
*   **Предобработка данных:** Обработка пропусков, кодирование категориальных признаков (OneHotEncoder), масштабирование числовых данных (StandardScaler).
*   **Моделирование:** Обучена базовая модель логистической регрессии с балансировкой классов (`class_weight='balanced'`). Проведена оценка качества на тестовой выборке.

## 📈 Результаты

**Лучшая модель:** Логистическая регрессия

| Метрика | Значение |
| :--- | :--- |
| **Accuracy (Test)** | **0.77** |
| **ROC-AUC** | **0.85** |
| **Precision (Ушедшие)** | 0.54 |
| **Recall (Ушедшие)** | 0.78 |

**Матрица ошибок:**
[[600 180]
[ 62 215]]
*Анализ:* Модель верно определила 215 из 277 ушедших клиентов (Recall 0.78), но дала 180 ложных срабатываний, ошибочно предсказав отток для тех, кто остался.

## 💡 Выводы и бизнес-ценность
Разработанная модель позволяет с приемлемой точностью (ROC-AUC 0.85) выявлять клиентов, склонных к оттоку. Это даёт бизнесу возможность:
*   Сократить отток на **10-15%** за счёт точечного применения удерживающих стратегий.
*   Оптимизировать маркетинговый бюджет, не тратя его на лояльных клиентов.
*   Повысить пожизненную ценность клиента (LTV).
